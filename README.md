# FaceBeam: Facial Recognition Attendance System

A professional facial recognition system built with **Python**, **OpenCV**, and **Flask** to automate identity verification and attendance tracking.

### 🚀 Features
- **Real-time Face Detection:** Uses OpenCV for high-speed identity verification.
- **Secure Backend:** Flask-based web application with **SQLite** database integration.
- **User Management:** Easy interface to add new students and log attendance.

### 🛠️ Tech Stack
- **Languages:** Python
- **Database:** SQLite (Facebeam.db)
- **Frameworks:** Flask, OpenCV, NumPy
